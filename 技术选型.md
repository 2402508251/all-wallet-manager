# 技术选型：Python + Vue3（PyWebView 嵌入式桌面）

### 1 后端 Python（核心业务、数据处理、本地 IO）

| 模块 | 技术方案 | 说明 |
|:---|:---|:---|
| 桌面容器 | pywebview | 嵌入 Vue 静态页面，进程内 JS/Python 互通，不暴露本地端口 |
| 数据库 | sqlite3 | 内置单文件本地库，无需服务；启用 `PRAGMA foreign_keys = ON` 强制外键约束 |
| 邮件 IMAP | imaplib + email | 标准库；附件下载、邮件过滤；**强制使用 SSL/TLS（端口 993）**，禁止明文传输授权码 |
| 表格解析 | openpyxl + csv | 解析微信 / 支付宝 xlsx/csv 账单 |
| 表格解析 | xlrd | 解析建设银行 .xls 账单 |
| 压缩处理 | zipfile | 标准 ZIP 解压 |
| 压缩处理 | pyzipper | AES 加密 ZIP 解压（建设银行账单专用），解析后自动清理临时文件 |
| 数据清洗 | pandas | 批量过滤、去重、金额/时间标准化 |
| 金额运算 | Python `Decimal` | 避免 float 浮点精度丢失 |
| 多线程 | threading | 耗时操作（邮箱同步、大文件解析）异步执行，防止 UI 卡死 |
| 打包工具 | Nuitka / PyInstaller | 打包 Windows 单 exe 程序 |


### 2 前端 Vue3（GUI 可视化界面）

| 模块 | 技术方案 | 说明 |
|:---|:---|:---|
| 构建工具 | Vite | 快速开发与构建 |
| UI 组件库 | Element Plus | 表单、弹窗、文件拖拽上传、分页表格 |
| 图表 | ECharts | 月度收支、资产负债统计、分类饼图、消费趋势柱状图 |
| 构建产物 | `vite build` → dist | 纯静态文件，由 pywebview 本地加载 |

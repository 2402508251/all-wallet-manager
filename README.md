# 本地统一账单工具

一个本地运行的多渠道统一账单管理与分析工具，支持微信、支付宝、建设银行等渠道账单的采集、解析、标准化、分类、对账和统计分析。

## ✨ 核心特性

- 🔒 **本地优先** - 数据保存在本机，适合对隐私和可控性要求更高的使用场景
- 🔄 **多渠道支持** - 统一接入微信、支付宝、建设银行等多渠道流水
- 📊 **完整处理链路** - 从采集、解析、标准化、分类、归属到对账、报表一体化打通
- 🛠️ **持续治理** - 支持分类修正、枚举映射、合并和快照回滚
- 💾 **长期维护** - 数据库采用重建优先策略，配套初始化脚本便于保持一致性
- 🖥️ **桌面应用** - 采用 PyWebView 桌面容器，启动体验更接近独立本地工具

## 📦 技术栈

**后端**
- Python 3.x
- PyWebView (桌面容器)
- SQLite (本地数据库)
- Pandas, openpyxl, xlrd (表格解析)
- pycryptodome (加密支持)

**前端**
- Vue 3.4+
- Vite 5.4+
- Element Plus 2.7+
- ECharts 5.5+
- Pinia (状态管理)
- Vue Router

## 🚀 快速开始

### 方式一：直接运行（推荐）

双击 `main.exe` 启动应用

### 方式二：源码运行

**环境要求**
- Python 3.8+
- Node.js 16+

**安装依赖**

```bash
# 安装 Python 依赖
pip install -r requirements.txt

# 安装前端依赖并构建
cd frontend
npm install
npm run build
cd ..
```

**启动应用**

```bash
python main.py
```

**数据库初始化**

初始化脚本位于 `scripts/init_db.sql`

## 📋 功能模块

| 模块 | 功能说明 |
|------|---------|
| 账单采集 | 支持文件导入、邮件收集等方式 |
| 账单解析 | 解析微信、支付宝、建设银行等渠道账单 |
| 标准化处理 | 统一字段、金额、时间、交易类型 |
| 分类管理 | 基于关键词和规则自动分类 |
| 账户与角色归属 | 支持账户识别、角色归属、家庭归档 |
| 对账与合并 | 支持跨平台合并、转账配对、信用卡流水识别 |
| 报表分析 | 提供看板、趋势、统计和明细查询 |
| 数据快照 | 支持变更快照与回滚 |
| 配置管理 | 支持分类规则、枚举映射、邮箱白名单等本地配置 |
